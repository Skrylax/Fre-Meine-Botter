Blue VS Red

Blue VS Red is a one lane "push the payload" MOBA game I worked on for a school project. Focusing on UI elements and player abilities.

  • GitHub: Blue VS Red
  • Environment: Unity3D

Infiltrator

Infiltrator is a 1 player dungeon crawler. During this project I mainly worked on randomly generating the dungeon floors and boss mechanics.

  • GitHub: Infiltrator
  • Environment: Unity3D

Crossroad Simulation "De Schrans @Leeuwarden - The Netherlands"

A two man project I worked on where one person build the Simulation and the other did the Controller. During this project I focused on making the traffic simulation as realistic as possible.

  • Source: Crossroad Simulation
  • Environment: Unity3D

Paint Application - Design Patterns

A six step school project, where in each step the application got extra functionality. Each of these steps had to be implemented using a specific design pattern.

The following design patterns are used in this application:

Command, Composite, Visitor, Strategy, Singleton and Decorator.

  • GitHub: Paint Application
  • Language: C#
